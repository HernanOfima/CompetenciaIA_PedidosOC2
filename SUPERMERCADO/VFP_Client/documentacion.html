📋 Resumen General

El Sistema Gestión OC2 es una aplicación cliente desarrollada en Visual FoxPro que consume servicios REST para la gestión integral de pedidos, facturación e inventario. El sistema implementa autenticación JWT y proporciona una interfaz moderna para operaciones comerciales.

🔐 Módulo de Login

Funcionalidad

El módulo de login proporciona autenticación segura mediante credenciales de usuario y genera tokens JWT para sesiones autenticadas.

1
Ingreso de Credenciales: El usuario introduce email y contraseña en el formulario de acceso.
2
Validación API: Las credenciales se envían al endpoint de autenticación vía HTTPS.
3
Token JWT: Si la autenticación es exitosa, se recibe y almacena el token de sesión.
4
Redirección: El usuario es dirigido al dashboard principal del sistema.

Implementación Técnica

Archivo: login.prg
Clase Principal: frmLogin
Protocolo: HTTPS REST API
Autenticación: JWT Bearer Token
* Ejemplo de autenticación loHTTP = CREATEOBJECT("WinHttp.WinHttpRequest.5.1") loHTTP.Open("POST", gcAPIBaseURL + "/auth/login", .F.) loHTTP.SetRequestHeader("Content-Type", "application/json") lcJSON = '{"email":"' + ALLTRIM(THIS.txtEmail.Value) + '","password":"' + ALLTRIM(THIS.txtPassword.Value) + '"}' loHTTP.Send(lcJSON)

🚀 Instalación y Configuración

Pasos de Instalación

1
Copiar Archivos: Extraer todos los archivos .prg a una carpeta local del sistema.
2
Verificar API: Asegurar que el backend esté ejecutándose en https://localhost:7109
3
Ejecutar Login: Desde Visual FoxPro, ejecutar DO login.prg para iniciar la aplicación.
4
Autenticarse: Usar las credenciales de prueba para acceder al sistema.

Configuración de API

Para cambiar la URL base de la API, modificar la variable en login.prg:

gcAPIBaseURL = "https://tu-servidor.com/api/v1"

Solución de Problemas Comunes

Error de Conexión HTTP

Verificar que WinHTTP esté disponible en Windows y que la API esté accesible.

Error 401 Unauthorized

Token JWT expirado o inválido. Realizar login nuevamente para obtener token fresco.

Error de Certificados SSL

En desarrollo, verificar que el certificado SSL del servidor sea válido o esté en modo desarrollo.

Problemas de Encoding

Asegurar que los caracteres especiales estén correctamente codificados en UTF-8.
